# CSG
FUJITSU Software Cloud Storage Gateway。
オンプレミス側にデプロイする仮想アプライアンスであり、
ローカルで受けたデータをクラウドのオブジェクトストレージへ連携する。
暗号化・ブロック単位の重複排除・圧縮を行う

## 今回実施した主なAPI作業
 - WSLでコマンド入力、インタネット接続できればなんでもよい
 - トークン認証、グループ、グループポリシー、ユーザ
 - AccessKey/SecretKeyの払い出し
 - バケット作成

### Swift
 - OpenStack 系のオブジェクトストレージの仕組み（独自 API を持つサービス）
### S3 
 - Amazon が開発したオブジェクトストレージ API
 - 多くのクラウド製品が採用する 事実上の業界標準 API

 ## 問題点
原因は S3 側で設定したグループポリシーが過度に制限されていた
手順では必要最小限の操作権限だけ許可していたが、
CSG がクラウドへ書き込みを行うためにはより広いS3 権限が必要だった。結果として、ポリシーを広げ（全許可）CSG との認証・書き込みが正常に動作した。
